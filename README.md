# Cultural Heritage Knowledge Graph

A semantic knowledge graph for cultural heritage and archaeological data, developed as part of my Diploma Thesis in Computer Engineering and Electronic Systems.

The project focuses on modeling archaeological information using Semantic Web technologies and domain ontologies to enable semantic interoperability, data integration, and advanced querying capabilities.

## Project Overview

This project involved:

- Designing and developing a cultural heritage ontology.
- Transforming structured data into RDF triples.
- Reusing established ontologies and standards for semantic interoperability.
- Building a knowledge graph for archaeological and cultural heritage information.
- Querying the graph using SPARQL.

## Technologies Used

- RDF (Resource Description Framework)
- Turtle (TTL)
- SPARQL
- Protégé
- Knowledge Graphs
- Semantic Web Technologies
- Ontology Engineering

## Ontologies and Standards

- CIDOC CRM
- CRMarchaeo
- GeoSPARQL
- SKOS
- PROV-O
- DCAT
